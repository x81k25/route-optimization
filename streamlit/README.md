# Route Optimization Streamlit Dashboard

This Streamlit app recreates the functionality of the HTML reports (aggregate_report.html and zone_x_report.html) in an interactive web dashboard.

## Features

### Aggregate Report
- **Summary Statistics**: Total zones, average weekly duration, utilization, POS time, and drive time
- **Zone-Level Metrics Table**: Interactive table with all zone metrics including:
  - Primary and secondary location counts
  - Weekly duration and utilization percentage
  - Over/under-utilized days
  - Total POS and drive time
- **All Zones Map**: Interactive Plotly map showing all zones with primary (star) and secondary (circle) locations

### Zone Details
- **Zone Selection**: Dropdown to select specific zones
- **Zone Summary**: Key metrics for the selected zone
- **Daily Itinerary**: Detailed breakdown of daily routes
- **Zone Map**: Zone-specific visualization (placeholder for now)

## Installation

1. Install and sync dependencies:
   ```bash
   uv sync
   ```

2. Or add individual dependencies:
   ```bash
   uv add streamlit pandas polars plotly pyyaml
   ```

## Running the App

### Option 1: Using the launcher script
```bash
./streamlit/launch.sh
```

### Option 2: Using the Python runner
```bash
uv run python streamlit/run.py
```

### Option 3: Direct streamlit command
```bash
uv run streamlit run streamlit/app.py --theme.base dark
```

## Data Requirements

The app expects the following files to exist:
- `output/itinerary.jsonl` - Route optimization results
- `output/aggregate_summary.jsonl` - Pre-calculated summary statistics (REQUIRED)
- `data/locations.jsonl` - Location details for map display
- `config/model-params.yaml` - Configuration parameters

### Data Flow
The app reads directly from the raw data files generated by the pipeline:
- Summary statistics are loaded from `aggregate_summary.jsonl` (not recalculated)
- Zone metrics are calculated from the `itinerary.jsonl` data
- Changes to the raw data files automatically flow through when the app is refreshed

## Dark Theme

The app includes custom CSS styling to match the dark theme of the original HTML reports, with:
- Dark background (#1a1a1a)
- Light text (#e0e0e0)
- Styled containers and metrics cards
- Blue accent colors (#4a9eff)

## Navigation

Use the sidebar to switch between:
- **Aggregate Report**: Overview of all zones
- **Zone Details**: Detailed view of individual zones

## Technical Details

- Built with Streamlit 1.28+
- Uses Polars for efficient data processing
- Plotly for interactive maps and visualizations
- Caches data and calculations for performance
- Responsive layout that works on different screen sizes
